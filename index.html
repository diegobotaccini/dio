Ferramentas Utilizadas (VSCode)

O Visual Studio Code é uma das ferramentas mais utilizadas para edição de código HTML. Ele oferece recursos como realce de sintaxe, autocompletar, embelezamento de código e extensões úteis, como o Live Server, que permite visualizar em tempo real as alterações no navegador.


Inspetor de Elementos (Browser)

O Inspetor de Elementos é uma funcionalidade dos navegadores que permite analisar e editar o código HTML diretamente na página exibida. Ele é útil para identificar elementos específicos, testar alterações no layout e entender melhor como o navegador interpreta o código.


Estrutura Básica do HTML

A estrutura básica de um documento HTML inclui a declaração !DOCTYPE html, a tag html, seguida pelas seções head (metadados, links a scripts e estilos) e body (conteúdo visível). Essa organização é fundamental para garantir que os navegadores interpretem corretamente o conteúdo.


Falando sobre Tags

Tags são os blocos de construção do HTML, usadas para criar e organizar elementos na página. Cada tag geralmente possui uma abertura tag e um fechamento /tag, definindo o início e o fim de um elemento, como p para parágrafos ou h1 para títulos.


Atributos das Tags

Atributos são informações adicionais que personalizam as tags. Eles aparecem dentro da tag de abertura e seguem o formato nome="valor". Exemplos incluem class para estilização e href para definir links, permitindo maior controle sobre os elementos.


Textos

O HTML oferece várias tags para trabalhar com textos, como p para parágrafos, h1 a h6 para títulos hierárquicos e span para estilizar partes específicas. A formatação básica é simples e eficaz para estruturar o conteúdo textual.


Listas Ordenadas e Não Ordenadas

As listas em HTML podem ser ordenadas ol ou não ordenadas ul, com itens marcados por li. Listas ordenadas são numeradas, enquanto as não ordenadas usam marcadores. Ambas são úteis para organizar informações.


Links

Links são criados com a tag a e o atributo href, que especifica o destino do link. Eles podem levar a outras páginas, seções dentro do mesmo documento ou até mesmo a arquivos externos, sendo um recurso essencial para a navegação na web.
